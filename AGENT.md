# ファイナンス運用メモ

## プロジェクトの目的
- この Codex プロジェクトで、個人および家族の資産・収入・支出を管理する。
- 家計キャッシュフローの改善と中長期の資産形成につながる、実務的なFPアドバイスを行う。

## 現在の最優先事項
- まずは支出の最適化から着手する。
- クレジットカード利用履歴データと銀行引落明細データを用いて、支出パターンを分析する。

## データ方針（初期）
- 個人支出・家族支出は、クレジットカード明細に加えて銀行口座引落も対象とする。
- 支出分析の主データソースは、クレジットカード利用履歴と銀行明細とする。

## 分析ワークフロー（支出見直し）
1. 対象カードの利用履歴ファイルと銀行入出金明細ファイルを収集する。
2. データ項目を標準化する（利用日、金額、加盟店、カテゴリ、利用者、メモ）。
3. 支出を分類する。
   - 個人 / 家族
   - 固定費 / 変動費
   - 必要支出 / 裁量支出
4. 最適化余地を特定する。
   - 高支出カテゴリ
   - 定期課金・サブスクリプション
   - 重複・不要の可能性がある支出
5. 月次インパクト見込みを添えて、実行可能な改善提案を作成する。

## 成果物
- カテゴリ別の月次支出サマリー
- 個人 / 家族の支出内訳
- 削減・圧縮候補（優先順位付き）
- 翌月の調整アクションプラン
